# 第一次自幹CPU就上手

## 動機

某天在看 jserv 的直播《你所不知道的C語言》聽到 jserv 說，資工系都有教計算機組織跟作業系統，所以理論上每個資工系畢業應該都要可以弄一個CPU跟OS出來。所以為了當一名~~合格~~的資工系畢業學生，只好找本書來參考，寫一個軟核心的CPU，並且嘗試porting一個作業系統到FPGA板子上。
